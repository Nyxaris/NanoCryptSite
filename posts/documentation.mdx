## Документация NanoCrypt

### Описание программного обеспечения

**NanoCrypt - это программное обеспечение, разработанное для безопасного хранения и управления паролями. Оно обеспечивает пользователей надежным и удобным способом сохранения и получения паролей для различных сервисов. С помощью NanoCrypt пользователи могут:**

- Регистрироваться и аутентифицироваться в системе.
- Сохранять пароли для различных сервисов с использованием шифрования.
- Получать сохраненные пароли для сервисов.
- Изменять пароли для отдельных сервисов.
- Блокировать аккаунт для дополнительной безопасности.

### Функциональные характеристики

1. **Регистрация аккаунта:** Пользователи могут создать новый аккаунт, указав имя, логин, пароль и секретное слово. При регистрации проверяются требования к паролю, такие как его длина и наличие различных типов символов.
2. **Аутентификация:** Пользователи могут войти в систему, предоставив свой логин и пароль. Система проверяет правильность учетных данных и разрешает доступ к функциям программы.
3. **Сохранение паролей:** Пользователи могут сохранять пароли для различных сервисов, указывая название сервиса и пароль. Пароли шифруются с использованием ключа шифрования и сохраняются в базе данных.
4. **Получение паролей:** Пользователи могут получать сохраненные пароли для сервисов, указывая название сервиса. Пароли расшифровываются с использованием ключа шифрования и отображаются пользователю.
5. **Смена пароля:** Пользователи могут изменить пароль для определенного сервиса, указав новый пароль. Изменение пароля влияет только на конкретный сервис, остальные сохраненные пароли остаются без изменений.
6. **Блокировка аккаунта:** Пользователи могут заблокировать свой аккаунт, требуя повторной аутентификации для доступа. Это обеспечивает дополнительный уровень безопасности, если пользователь покидает устройство или программу на некоторое время.
